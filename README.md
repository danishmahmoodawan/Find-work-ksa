# Find-work-ksa
Browse a comprehensive database of verified job listings across Saudi Arabia. Our free portal helps you filter by industry and city to find the perfect match for your career goals.

# FindWork KSA 🇸🇦

A Job Board Web Application for the Saudi Arabian market built with Python and Flask.

## Live Website
🌐 [findworkksa.com](https://findworkksa.com)

## Tech Stack
- Python 3.14
- Flask
- SQLAlchemy
- SQLite
- Bootstrap 5
- HTML/CSS

## Features
- Browse job listings
- Search jobs by title
- Filter by category
- Post new jobs
- View full job details
- Delete jobs
- Responsive design

## Installation

```bash
# Clone the repo
git clone https://github.com/danishmahmoodawan/Find-work-ksa

# Install dependencies
pip install flask flask-sqlalchemy

# Run the app
python app.py
```
